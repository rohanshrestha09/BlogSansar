## Installation

## Running Server

```bash
  git clone https://github.com/rohanshrestha09/BlogSansar.git
  cd BlogSansar
  npm install
  npm run start:dev
```

## Running Client

```bash
  git clone https://github.com/rohanshrestha09/BlogSansar.git
  cd BlogSansar/client
  npm install
  npm run start
```
